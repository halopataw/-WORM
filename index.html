Guardian Manifesto // Action Protocol

【 前言 】

今天，你不只是來海邊玩的人。你也是這片沙灘的短暫守護者。
你的每一個選擇，都會決定海灘是更乾淨、更安全，還是更脆弱。

1. 成為守護者

我願意用尊重的方式接近海岸，用觀察代替打擾，用理解代替佔有，讓今天的到來，不成為這片沙灘的負擔。

2. 海岸之註記

海的禮貌，是看見而不打擾。石頭翻得起，也要放得回。不是每個發現都要帶走。最好的紀念品是學會守護的方法。

3. 自然的提醒

海邊的天氣，說變就變。浪看起來平靜，不代表沒有力量。退潮讓人更靠近海岸，也可能讓人忽略回程的時間。風、浪、雨、潮汐，都是自然給我們的提醒。來到海邊，先尊重大海的節奏，再享受沙灘的美好。

4. 安全的警示

出發前，先看天氣也看海況。風太大、浪太高、天色轉暗時，不要勉強停留。不要獨自走向太靠近海浪的地方，也不要小看濕滑礁岩的危險。最安靜的海面，也藏著最需要小心的力量。平安回家最重要。

5. 離開前，留下一個更好的沙灘

帶走自己的垃圾，也願意多撿起一件不屬於海岸的東西。不留下破壞，不帶走生命。當你離開時，讓這片沙灘比你來的時候更乾淨、更安全，也更值得下一個人好好珍惜。
